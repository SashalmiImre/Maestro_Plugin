## Új ismeret / felfedezés

- A `workflowLatestUpdatedAtRef` globális `$updatedAt` Map mintája más Realtime-érzékeny dual-list-re is alkalmazható (pl. Phase 2 publikációk archív/aktív, ha valaha kell).
- A `useOrgRole` hookban szándékos a default `activeOrganizationId` hiánya — 3 szemantikai variáns (active-org / workflow-owner-org / publication-org) miatt implicit default elrejtené a választást.
- A `getDatabases()` / `getFunctions()` modul-szintű export érdemi különbség a Provider-szintű `useData().databases`-tól: az előbbi singleton, az utóbbi a Provider lifecycle-ja alatt él (scope-reset miatt megmarad). Funkcionálisan ekvivalens.
